# Rename-File
用 Python 批量修改文件名

### 0:05, Feb, 27th, 2018 @ dormitory, B5 R627

## 使用原理

用 OS 模块的 listdir 方法枚举目录下的文件名，用列表切片过滤后缀，再用 rename 方法重命名文件
